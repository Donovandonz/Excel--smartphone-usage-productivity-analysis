## Project Overview
This project analyzes smartphone usage patterns and their impact on productivity, stress levels, and daily habits. The analysis examines correlations between device types, occupations, age groups, and various metrics including screen time, stress levels, and caffeine consumption.

---

## 🛠️ Tools Used

- **Microsoft Excel**
  - Pivot Tables for data aggregation
  - Pivot Charts for visualization
  - Slicers for interactive filtering
  - Conditional Formatting for highlighting trends
  - Formulas for calculations (SUMIFS, COUNTIFS, AVERAGE, etc.)

## 💡 Key Insights

### 1. Platform Parity Confirmed
> **Insight**: The 0.04 hour (2.4 minute) difference between Android and iOS users confirms that **operating system choice has minimal impact on daily screen time**.

**Implication**: App developers and marketers should continue treating both platforms equally.

### 2. Revised Normal Range
> **Insight**: The 5.43-6.53 hour range represents **healthy, typical smartphone usage** for this demographic.

**Implication**: Users falling within this range (85-90% of users) represent "average consumers."

### 3. Android Slight Edge
> **Insight**: Android users show marginally higher engagement (6.53 vs 6.49 hours).

**Possible Explanations**:
- Android's customization options encourage more interaction
- Different demographic profiles (Android users may skew younger)
- More affordable devices = longer ownership/usage
- Background processes keeping screens active

## 💡 Key Insights

### 1. Gender Parity in Stress
> **Insight**: Unlike many studies that show significant stress disparities, this dataset reveals **virtually identical stress levels** across all genders.

**Implication**: External factors (occupation, age, screen time) likely play a larger role in stress than gender alone.

### 2. The 5.5 Benchmark
> **Insight**: An average stress level of **5.5** on what appears to be a 1-10 scale represents **moderate stress**.

**Interpretation**: 
- Below 5: Low stress
- **5.5: Moderate stress (current average)**
- Above 7: High stress (requires attention)

### 3. Population Uniformity
> **Insight**: The 0.01 point spread suggests remarkable uniformity in how different genders experience and report stress.

**Implication**: Wellness programs can be designed for the general population rather than gender-specific approaches.
